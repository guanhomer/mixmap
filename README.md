# MIXMAP
**MIXture Methylation Analysis in Plasma**

mixmap provides tools for extracting and analyzing fragment-level CpG methylation patterns from bisulfite sequencing data. It targets heterogeneous DNA mixtures (e.g., cfDNA) where per-fragment methylation signals are informative.

---

## Components

### 1. `per_read_per_cpg.py`
Extracts per-read, per-CpG methylation calls from BAM files and collapses them to fragment-level observations.

- Strand-aware calling (Bismark / bwa-meth compatible)
- External sorting for scalability
- Conflict resolution within fragments

### 2. `logsum.R`
Computes fragment-level log-likelihood sums against a reference methylation matrix.

- Designed for SLURM array execution
- Maps CpGs to reference probes
- Computes log P(data | reference) per fragment
- Aggregates across CpGs within each fragment

---

## Workflow

1. **Extraction**
```bash
python per_read_per_cpg.py ref.fa input.bam output.tsv
```

2. **Likelihood computation**
```bash
Rscript logsum_array.R
```

## Requirements

- Python ≥ 3.8
- pysam
- R ≥ 4.0
- data.table (R)
- Unix `sort`

---

## Input formats

### Per-read CpG file
Generated by `per_read_per_cpg.py`

Columns:
- fragment_id
- chrom
- ref_pos
- strand
- call (m/u)
- fragment_span

### Reference matrix

- Rows: CpG IDs (`chrom_position`)
- Columns: reference methylation profiles
- Values: probabilities in [0, 1]

---

## Output

### Extraction output
| Column | Description |
|--------|------------|
| fragment_id | Fragment index |
| chrom | Chromosome |
| ref_pos | CpG position |
| strand | + / - |
| call | m / u |
| fragment_span | Fragment length |

### Log-sum output
| Column | Description |
|--------|------------|
| pos_id | First CpG ID in fragment |
| fragment_span | Fragment length |
| n_probe | Number of CpGs |
| n_probe_meth | Number methylated |
| ... | Log-probabilities per reference |

---

## Notes

- CpGs are mapped using strand-adjusted coordinates
- Fragments with <3 CpGs are excluded
- Probabilities are clipped to avoid log(0)
- Designed for large-scale BAM processing via streaming + external sort
